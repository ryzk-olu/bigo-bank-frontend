# BigO-Bank Frontend

A Single Page Application (SPA) that simulates an online banking client. The project is built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**, following a modular architecture and prepared for future integration with a **FastAPI** backend.

## Table of Contents

* [Features](#features)
* [Tech Stack](#tech-stack)
* [Getting Started](#getting-started)

---

## Features

* Secure authentication with JWT 
* Protected routes
* Dashboard with customer accounts
* Multi-currency account management
* Exchange rates widget
* Persistent user session using Local Storage
* Service layer prepared for REST API integration

---

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS v4
* React Router DOM
* Context API

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
